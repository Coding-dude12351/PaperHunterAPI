# PaperHunt REST-API

# Project Overview and Goals
- The goal is to develop a robust, secure, and user-friendly cross-platform application (Web and Mobile) that provides JCE and MSCE students access to MANEB past papers and offers a unique, contextualized AI chat tutor to answer questions based on the paper content.

# Key Objectives
## 1. Accessibility 
- Provide easy access to structured past papers (categorized by Level, Subject, and Year).
## 2.	Offline Capability
- Allow mobile users to download and view papers without an active internet connection.
## 3.	Innovation
- Implement a Retrieval-Augmented Generation (RAG) system to enable the AI Chat Tutor.
## 4.	Security
- Ensure secure content delivery and user authentication.
